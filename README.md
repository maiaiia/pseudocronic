# pseudocronic

## Sintaxa Pseudocod

### Tipuri de Date
- bool 
- int
- float / double
- string
 
### Operatii
#### Operatii Logice
- `=` - egalitatea
- `!=` - neegalitatea
- `<` - mai mic
- `<=` - mai mic sau egal
- `>` - mai mare
- `>=` - mai mare sau egal
- `sqrt()` - radacina patrata

#### Operatii Aritmetice 
- `+` - adunare
- `-` - scadere
- `*` - inmultire 
- `\` - impartire
- `[\]` sau `DIV` - impartire cu partea intreaga 
- `%` sau `MOD` - modulo

### Structuri Repetitive 

#### pentru, executa
```
 -pentru <variabila> <- <expresie initiala>, <expresie finala>, [pas] executa
|   <instructiuni>
 - stop pentru
```

#### cat timp, executa
```
 -cattimp <conditie> executa
|   <instructiuni>
 - stop cattimp
```

#### executa, cat timp

```
 -executa 
|   <instructiuni>
 - cattimp <conditie>
```

#### repeta, pana cand 

```
 - repeta
|   <instructiuni>
 - pana cand <conditie>
```

### Operatii Built-in

#### Citirea
`citeste <lista variabile>`

#### Afisarea 
`scrie <lista expresii>` 

#### Atribuirea
`<variabila> <- <expresie>`
